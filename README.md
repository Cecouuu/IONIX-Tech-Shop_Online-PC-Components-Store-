# IONIX Tech Shop

IONIX Tech Shop е уеб приложение за онлайн магазин за компютърни компоненти, разработено с PHP и MySQL. Проектът включва публична част за потребители, продуктов каталог, количка, поръчка като гост, потребителски профили и административен панел за управление на магазина.

## Основни функционалности

- Каталог с продукти, категории, цени, наличност и изображения
- Детайлна страница за всеки продукт
- Количка и завършване на поръчка
- Поръчка като регистриран потребител или като гост
- Регистрация, вход и потребителска сесия
- История на поръчките за регистрирани потребители
- Административен панел за продукти, поръчки, потребители, справки и промоции
- Седмични продукти и уикенд оферти на началната страница
- Автоматично създаване на база данни и начални данни при локално стартиране

## Използвани технологии

- PHP 8+
- MySQL или MariaDB
- HTML5
- CSS3
- Bootstrap 5
- XAMPP за локална среда

## Стартиране на проекта

1. Постави папката на проекта в `xampp/htdocs/Project`.
2. Стартирай `Apache` и `MySQL` от XAMPP.
3. Отвори сайта на адрес `http://localhost/Project/Main.php`.
4. Ако базата данни не съществува, приложението ще създаде нужните таблици автоматично.

Настройките за базата данни се намират в [Be/config/database.php](Be/config/database.php). По подразбиране проектът използва:

```env
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=project_app
DB_USER=root
DB_PASS=
```

## Demo Admin
